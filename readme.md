# Installing

```
$ yarn   # or npm install
```

(or *npm install* if you're using npm)

# Running locally

```
$ heroku local
```

# Deploying

```
$ heroku create
$ git push heroku master
$ heroku open # shows 'unnamed'

$ heroku config:set APP_NAME=MyName
$ heroku open # shows 'MyName'
```

# Linting

```
$ yarn run lint   # or npm run lint
```
